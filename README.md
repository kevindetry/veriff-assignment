# Veriff Assignment

## Install

To run the scripts below, you need to install the dependencies by running `yarn`.

## Available Scripts

In the project directory, you can run:

### `yarn dev`

Runs the app in the development mode.

### `yarn build`

Builds the app for production to the `dist` folder.

### `yarn preview`

Runs the app in the production mode. Requires running `yarn build` first.

### `yarn lint`

Checks the validity of the source files with `tsc` and `eslint`.

### `yarn test`

Launches the test runner.
